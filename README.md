# ?? Plataforma Omnichannel - Automatizacion CI/CD e Infraestructura Cloud (EFT)
Este repositorio contiene la solucion completa para la Evaluacion Final Transversal (EFT).
## ??? Arquitectura del Monorrepo
* **back-Ventas_SpringBoot**: Microservicio de ventas.
* **back-Despachos_SpringBoot**: Microservicio de despachos.
* **front_despacho**: Interfaz de usuario web.
* **terraform**: Codigo de Infraestructura en AWS.
* **.github/workflows**: Pipeline de automatizacion en GitHub Actions.
## ?? Entorno de Desarrollo Local
Para levantar todo el ecosistema localmente ejecuta: docker-compose up --build
